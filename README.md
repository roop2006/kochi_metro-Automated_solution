# kochi_metro-Automated_solution
Automated Document Workflow Platform for KMRL powered by AI: Documents in multiple languages and formats
idea/solution:
 Smooth overnight feed from IBM Maximo, SharePoint,
Emails, WhatsApp PDFs
 summarization and classification using NLP for officials
 Real-time notifications with auto-routing replaces silos and
manually administered routing.Central database provides memory of the institution and
improves retrieval.
 Additional QR code functionality has been added to link
paper records to the digital documents
problem resolution:
Eliminate manual document tracking and reduce
information delays.
 Ensure cross-department visibility with automated alerts
and routing
innovation:
Low-cost, mobile-first solution using easily
deployable QR codes.
 Tailored specifically for metro rail maintenance and
operation workflows.
 Built to scale and integrate future technologies like
IoT and AI.
AI & Backend: Python (Flask) for NLP,
classification, and workflow automation.
Frontend: React (web) / Flutter (mobile) for
intuitive staff interfaces and real-time
dashboards.
Database: PostgreSQL for managing document
metadata, user actions, and audit logs.
QR Module: Python qrcode for unique QR code
generation and decoding (as an add-on).
Cloud/Hosting: AWS for scalable, secure
deployment and centralized document access.
Integration: REST APIs for connecting with
Maximo, SharePoint, email, and WhatsApp.
